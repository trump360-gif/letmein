# 블랙라벨 (Black Label) PRD — 프레임 문서

> **버전**: v2.2 | **최종 수정**: 2026-03-30
> Flutter 모바일 앱 (iOS/Android) + Next.js 웹 CMS
> "렛미인 유튜브 출연자 스토리 + 코디네이터 매칭으로 검증된 프리미엄 성형 상담"

---

## 기술 스택 요약

| 영역 | 기술 |
|------|------|
| 앱 | Flutter + Riverpod |
| 백엔드 | Go (Gin) + PostgreSQL + Redis |
| 채팅 | Centrifugo + WebSocket |
| 파일 | Cloudflare R2 + CDN |
| CMS | Next.js + shadcn/ui |
| 인증 | JWT + 카카오 (MVP-B: Apple) |
| 인프라(MVP) | Docker Compose + Caddy |

## 핵심 제약사항

- 금액 기재 절대 금지 — 키워드 필터링 필수 (상담 + 채팅)
- 신체 사진 = 민감정보 — 별도 동의 필수
- 만 18세 미만 가입 차단
- 채팅 면책 고지 상시 표시
- **수익 모델 확정: "매칭은 무료, 수익은 광고에서만"** → [12_revenue.md](./12_revenue.md)
- 매칭 건당 수수료 영구 제외 (의료법 27조 위반)
- 유료 노출에 "광고" 라벨 필수 (의료법 56조 + 표시광고법)
- 의료법 27조 법률 의견서 출시 전 필수
