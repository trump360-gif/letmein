> 레퍼런스: 다이사(Daisa) 앱
> 기본 모드: 다크 | 프로젝트: Flutter + Riverpod

|------|------|

## 컬러 팔레트

### 다크 모드 (기본)

| 구분 | 컬러 | 용도 |
|------|------|------|
| 배경 (Primary) | `#0D0D0D` ~ `#1A1A1A` | 앱 전체 배경 |
| 표면 (Surface) | `#1E1E2E` ~ `#252535` | 카드, 바텀시트, 모달 |
| 포인트 (Accent) | `#C0392B` ~ `#A93226` | CTA 버튼, 핵심 인터랙션, 배지 |
| 서브 포인트 | `#D4A574` (골드/샴페인) | 프리미엄 강조, 출연자 배지 |
| 텍스트 (Primary) | `#F5F5F5` | 본문 텍스트 |
| 텍스트 (Secondary) | `#9E9E9E` | 보조 텍스트, 힌트 |
| 구분선 | `#2A2A3A` | 디바이더, 보더 |

### 라이트 모드 (옵션)

| 구분 | 컬러 | 용도 |
|------|------|------|
| 배경 | `#FAFAFA` | 앱 전체 배경 |
| 표면 | `#FFFFFF` | 카드, 모달 |
| 포인트 | `#C0392B` | CTA 버튼 (다크와 동일) |
| 텍스트 | `#1A1A1A` | 본문 텍스트 |

## 타이포그래피 (Pretendard)

| 요소 | 사이즈 | 웨이트 | 용도 |
|------|--------|--------|------|
| 대제목 | 24px | Light (300) | 히어로 타이틀, 섹션 헤더 |
| 소제목 | 18px | Medium (500) | 카드 타이틀, 탭 헤더 |
| 본문 | 14px | Regular (400) | 일반 텍스트 |
| 캡션 | 12px | Regular (400) | 보조 정보, 타임스탬프 |
| CTA 버튼 | 16px | SemiBold (600) | 버튼 텍스트 |

> 럭셔리 느낌을 위해 Light/Regular 웨이트를 적극 활용. Bold는 CTA와 핵심 수치에만 사용.

## 컴포넌트 스펙

### Card

| 속성 | 값 |
|------|-----|
| 배경 | Surface (`#1E1E2E`) |
| 모서리 | 16px radius |
| 그림자 | 없음 (다크 모드에서는 보더로 구분) |
| 보더 | 1px solid `#2A2A3A` |
| 내부 패딩 | 16px |

### CTA Button

| 속성 | 값 |
|------|-----|
| 배경 | Accent (`#C0392B`) |
| 텍스트 | White |
| 모서리 | 12px radius |
| 높이 | 52px |
| 눌림 효과 | 밝기 -10% |
| 비활성 | opacity 0.4 |

### Chip (선택 태그)

| 속성 | 값 |
|------|-----|
| 기본 | 보더만 (`#2A2A3A`), 텍스트 Secondary |
| 선택됨 | 배경 Accent, 텍스트 White |
| 모서리 | 20px radius (pill 형태) |

### Input (텍스트 입력)

| 속성 | 값 |
|------|-----|
| 배경 | Surface (`#1E1E2E`) |
| 보더 | 1px solid `#2A2A3A`, 포커스 시 Accent |
| 텍스트 | Primary (`#F5F5F5`) |
| 플레이스홀더 | Secondary (`#9E9E9E`) |
| 모서리 | 12px radius |

### Avatar (프로필)

| 속성 | 값 |
|------|-----|
| 기본 | 원형, 40px |
| 스토리 바 | 원형, 64px |
| 새 스토리 테두리 | 2px solid Accent (`#C0392B`) |
| 읽은 스토리 테두리 | 2px solid `#2A2A3A` |

### Bottom Navigation

| 속성 | 값 |
|------|-----|
| 배경 | `#0D0D0D` |
| 활성 아이콘 | Accent (`#C0392B`) |
| 비활성 아이콘 | `#666666` |
| 구분선 | 상단 1px `#2A2A3A` |

## UI 규칙

### 필수 패턴

| 규칙 | 설명 |
|------|------|
| 드롭다운 사용 금지 | 탭 또는 칩으로 대체 |
| 아코디언 사용 금지 | 항상 펼쳐진 상태로 노출 |
| 순서 변경 | 드래그 앤 드롭으로 구현 |
| 텍스트 입력 최소화 | 탭/칩 선택 중심으로 설계 |

### 트랜지션

| 요소 | 효과 |
|------|------|
| 화면 전환 | fade + slide 복합 |
| 카드 탭 | 스케일 애니메이션 (0.98 -> 1.0) |
| 로딩 | 스켈레톤 UI (shimmer, 다크 테마 맞춤) |
| 히어로 스크롤 | 패럴랙스 효과 |

### 이미지 가공

- 유튜브 썸네일을 그대로 사용하지 않음 — 다크 오버레이 + 타이틀 재배치
- 이미지 위 텍스트는 항상 그림자/그래디언트 처리로 가독성 확보

## 테마 토글 (MVP-B)

| 항목 | 내용 |
|------|------|
| 위치 | 마이페이지 > 설정 > 테마 |
| 옵션 | 다크 (기본) / 라이트 / 시스템 설정 따르기 |
| 저장 | SharedPreferences 로컬 저장 |
| 반영 | 선택 즉시 전체 앱에 반영 (재시작 불필요) |
| 현재 상태 | `AppTheme.light` 정의 완료, `ThemeMode.dark` 하드코딩 상태 -- 토글 연결 필요 |

**AC**
- [ ] 설정에서 다크/라이트/시스템 3가지 옵션 선택 가능
- [ ] 선택 즉시 테마 반영
- [ ] 앱 재시작 후에도 선택 유지
- [ ] 시스템 설정 변경 시 자동 반영

## Spacing System (AppSpacing)

> Flutter `app_theme.dart`에서 상수로 관리

| 상수명 | 값 | 용도 |
|--------|-----|------|
| `xxs` | 2px | 미세 간격 |
| `xs` | 4px | 아이콘-텍스트 간격 |
| `sm` | 8px | 칩 내부 패딩, 리스트 아이템 간격 |
| `md` | 12px | 카드 내부 요소 간격 |
| `base` | 16px | 카드 패딩, 섹션 간격 |
| `lg` | 20px | 섹션 헤더 마진 |
| `xl` | 24px | 화면 좌우 패딩 |
| `xxl` | 32px | 대섹션 간격 |
| `xxxl` | 48px | 히어로-콘텐츠 간격 |

## Flutter 구현 노트

### v1.4 -> v2.0 마이그레이션

| 항목 | v1.4 | v2.0 |
|------|------|------|
| 기본 모드 | 라이트 | **다크** |
| Primary 컬러 | `#6C5CE7` (퍼플) | `#C0392B` (레드/버건디) |
| 배경 | `#FFFFFF` | `#0D0D0D` |
| 카드 배경 | `#F8F9FA` | `#1E1E2E` |
| 폰트 웨이트 | Medium 중심 | Light/Regular 중심 |

### 테마 구현

- `app_theme.dart`에서 `ThemeData` 다크 모드 우선 설정
- `ColorScheme.dark()` 기반으로 커스텀 팔레트 적용
- 기존 하드코딩된 컬러값을 테마 토큰으로 교체
- `AppSpacing` 상수 클래스로 간격 관리
