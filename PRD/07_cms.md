## 신고 처리 워크플로우

신고 접수 → 24시간 내 확인 → 블라인드 또는 기각 → 당사자 알림

| 신고 사유 | 처리 기준 |
|----------|----------|
| 금액 직접 기재 | 경고 / 2회 시 계정 정지 |
| 외부 연락처 유도 | 경고 / 2회 시 계정 정지 |
| 타 병원 비방 | 즉시 블라인드 후 검토 |
| 허위 정보 | 확인 후 블라인드 |
| 음란·불법 콘텐츠 | 즉시 블라인드 + 계정 정지 |
