# 블랙라벨 (Black Label) — Product Requirements Document

> **버전**: v2.2 | **최종 수정**: 2026-03-30
> "렛미인 유튜브 출연자 스토리 + 코디네이터 매칭으로 검증된 프리미엄 성형 상담"

---

## 서비스 비전

유튜브 **'렛미인'** 포맷의 변신 스토리 콘텐츠를 기반으로 유입된 팬덤을 자체 앱/웹 플랫폼으로 **락인(Lock-in)** 시키는 프리미엄 뷰티 콘텐츠·상담 플랫폼.

### 핵심 가치

- **탈(脫) 알고리즘 종속** — 유튜브 알고리즘 의존도에서 벗어나 자체 플랫폼 생태계 구축
- **투명한 정보 제공** — 영상으로 검증된 변신 콘텐츠 + 실제 경험 기반의 신뢰 있는 정보
- **코디네이터 매칭** — 전문 코디네이터가 검토 후 맞춤 프리미엄 병원을 연결

### 핵심 플로우

```
유튜브 팬덤 유입 → 출연자 스토리/커뮤니티 체류
→ 상담 접수 (탭 기반 7단계) → 코디네이터가 2~3개 병원 매칭
→ 매칭된 모든 병원과 1:1 채팅 → 유저가 직접 선택/방문
```

## 수익 모델 (확정)

### 핵심 원칙: "매칭은 무료, 수익은 광고에서만"

의료법 제27조(환자 유인·알선 금지) 분석 결과, **매칭 기능과 과금을 완전 분리**하는 구조를 채택.

| 상품 | 과금 | 시점 |
|------|------|------|
| 프리미엄 입점 | 월정액 — 상담 카드 정보 확장, 검색 상단 노출, 프로필 확장 | MVP-B |
| 네이티브 광고 | 충전형 CPM — 커뮤니티 피드 콘텐츠형 광고 | 3차 |

### 영구 제외 항목

- 매칭·상담 건당 수수료 (CPA) — 의료법 27조 위반 확실
- 과금 병원만 매칭 참여 가능한 구조 — 알선 대가 해석
- 유료 노출에 "광고" 라벨 필수 (의료법 56조 + 표시광고법)
- 계약서에 "소개/알선/매칭" 용어 절대 금지 — "광고 게재 서비스" 명목만 사용

### 수익화 로드맵

| 단계 | MAU 구간 | 수익원 |
|------|---------|--------|
| MVP-A | 0~2,000 | 없음 (무과금) |
| MVP-B | 2,000~10,000 | 프리미엄 입점 (월정액) |
| 3차 | 10,000~50,000 | + 네이티브 광고 (CPM) |
| 4차 | 50,000+ | + 데이터 리포트 (비식별 통계) |

## 기술 스택

| 영역 | 기술 | 비고 |
|------|------|------|
| 앱 | Flutter + Riverpod | iOS / Android 동시 대응 |
| 백엔드 | Go (Gin) + PostgreSQL + Redis | 고성능, 낮은 메모리 |
| 채팅 | Centrifugo + WebSocket | Redis pub/sub 브로커 |
| 파일 | Cloudflare R2 + CDN | S3 호환 API |
| CMS | Next.js (App Router) + shadcn/ui | 관리자 전용 |
| 인증 | JWT + 카카오 (MVP-B: Apple) | Refresh Token Redis 관리 |
| 인프라 (MVP) | Docker Compose + Caddy | 단일 서버 |
| 푸시 알림 | FCM (Firebase Cloud Messaging) | MVP-B |
| 모니터링 | Prometheus + Grafana / Sentry | 인프라 + 에러 트래킹 |

### 단계별 인프라 전략

| Phase | 규모 | 인프라 |
|-------|------|--------|
| Phase 1 (MVP) | 0~1만 유저 | Docker Compose + Caddy + 단일 PostgreSQL/Redis |
| Phase 2 | 1만~5만 유저 | Docker Swarm + Nginx + PgBouncer + Redis Sentinel |
| Phase 3 | 5만+ 동접 | Kubernetes + HPA + Redis Cluster + Elasticsearch |

## Out of Scope

- 비대면 화상 상담 내재화 (3차 이후)
- 카메라 가이드 프레임 (부위별 실루엣 오버레이)
- 유해 콘텐츠 자동 감지 (MVP-B 검토)
- 자체 비디오 호스팅 (MVP는 유튜브 딥링크)
- 유저 과금 모델 (현재 미검토)
