## 콜드스타트 전략 *(v2.0 갱신)*

> 렛미인 유튜브 채널 구독자를 핵심 유입 채널로 활용.

| 단계 | 전략 | 목표 |
|------|------|------|
| 출시 전 (D-4주) | 강남권 프리미엄 성형외과 10~20곳 사전 입점 영업 (무료, 직접 방문) | 병원 공급 확보 |
| 출시 전 (D-3주) | 렛미인 유튜브 출연자 10~20명 앱 가입 + 출연자 인증 | 출연자 콘텐츠 확보 |
| 출시 전 (D-2주) | 출연자가 앱 내 첫 스토리 등록 (회복 일지, Q&A 등) | 빈 피드 방지 |
| 출시 전 (D-1주) | 운영팀 시드 콘텐츠 20건 (Q&A + 비포&애프터 가이드) | 커뮤니티 시드 |
| 출시 시 | 렛미인 유튜브 채널에서 앱 출시 공지 + 유튜브 영상 내 앱 링크 | 팬덤 유입 |
| 출시 후 1개월 | 유튜브 + SNS 홍보, 레퍼럴 이벤트 오픈 | MAU 500 목표 |
| 출시 후 3개월 | 병원 50곳, 유저 피드백 반영 MVP-B 출시 | MAU 2,000 목표 |

## 유튜브 콘텐츠 히어로 (MVP-B)

**플랫폼 운영사가 직접 제작·운영하는 유튜브 채널의 영상** + 출연자 에피소드를 메인화면 히어로 영역에 노출.
병원 콘텐츠가 아닌 운영사 자체 콘텐츠이므로 의료광고 심의 대상 아님.
터치 시 유튜브 앱(미설치 시 브라우저)으로 이동.

| 항목 | 내용 |
|------|------|
| 콘텐츠 주체 | 플랫폼 운영사 자체 유튜브 채널 (렛미인) |
| 썸네일 | 플랫폼 톤앤매너에 맞게 **가공하여 노출** (그대로 사용하지 않음) → [13_design.md](./13_design.md) |
| 노출 위치 | 메인화면 히어로 영역 (상단 가로 슬라이드) |
| 터치 동작 | 유튜브 딥링크 |
| 영상 등록 | CMS에서 에피소드 등록 + 출연자 연결 → [07_cms.md](./07_cms.md), [12_cast_member.md](./12_cast_member.md) |
| 자동 슬라이드 | 5초 간격 (사용자 조작 시 일시 정지) |
| 레이아웃 | 가로 스크롤 슬라이드, 다크 오버레이 + 타이틀 재배치, 하단 인디케이터 도트 |

## 레퍼럴 이벤트 (MVP-B)

> ⚠️ 병원 방문·상담 완료를 트리거로 하는 레퍼럴은 의료법상 알선 대가로 해석될 수 있어 제외. → [10_legal.md](./10_legal.md)

| 항목 | 내용 |
|------|------|
| 구조 | 유저 A가 친구 B에게 추천인 코드 공유 → B가 앱 가입 → A·B 이벤트 자동 응모 |
| 트리거 | 피추천인 앱 가입 완료 |
| 보상 방식 | 추첨 (건수 비례 아닌 순수 추첨) |
| 보상 종류 | 상품권 (운영 정책으로 결정) |
| 중복 참여 | 추천 건수 누적 가능 (응모권 누적) |
| 금지 | 병원 방문·상담 완료 연동 트리거 |
