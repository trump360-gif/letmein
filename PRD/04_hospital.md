## 병원 가입 승인 프로세스

```
병원 앱 설치 → 사업자등록번호 입력 → 의료기관 개설 신고증 업로드
→ 병원 기본 정보 입력 → CMS 운영팀 수동 승인 (1~3영업일)
→ 승인 완료 알림 → 병원 계정 활성화
```

| 단계 | 상세 |
|------|------|
| 사업자등록번호 | 국세청 API 연동 유효성 검증 |
| 의료기관 개설 신고증 | 사진 업로드 → [07_cms.md](./07_cms.md) 에서 수동 확인 |
| 기본 정보 입력 | 병원명, 주소, 전화번호, 진료시간, 전문 시술 분야, 대표 이미지 |
| 승인 기준 | 사업자번호 유효 + 신고증 확인 + 실제 성형외과 여부 |
| 반려 시 | 사유 안내 + 재신청 가능 |

## 병원 탐색 — 구조

### 진입 경로 (A+B 듀얼)

- 하단 탭 "병원 탐색" 직접 진입 (A — 검색·필터 중심)
- 커뮤니티 비포&애프터 게시물의 병원 태그 탭 (B — 자연 유입) → [03_community.md](./03_community.md)

### 리스트 뷰 (MVP-A)

**검색 + 필터**

| 항목 | 내용 |
|------|------|
| 검색 | 병원명, 시술 분야 키워드 |
| 필터 | 시술 분야 (카테고리·세부 시술) / 지역 |
| 정렬 | 시술 분야 일치순 / 거리 가까운 순 / 태깅 후기 많은 순 |

**프리미엄 병원 상단 노출 (MVP-B)**

> 수익 모델 상세는 → [12_revenue.md](./12_revenue.md) 참조

| 항목 | 내용 |
|------|------|
| 노출 위치 | 검색 결과 최상단 (일반 결과 위) |
| 라벨 | **"광고"** 라벨 필수 표시 |
| 최대 노출 수 | 검색당 프리미엄 병원 최대 3개 |
| 노출 기준 | 프리미엄 병원 간 — 시술 분야 일치도 + 랜덤 로테이션 |
| 일반 병원 | 프리미엄 영역 아래에 기존 정렬 기준으로 노출 |

**병원 카드 노출 항목**

| 항목 | 무료 병원 | 프리미엄 병원 |
|------|----------|-------------|
| 병원명 + 대표 이미지 | ✅ | ✅ |
| 전문 시술 분야 | 칩 최대 3개 | 칩 최대 3개 |
| 위치·거리 | ✅ | ✅ |
| 태깅 후기 수 | ✅ | ✅ |
| 병원 사진 추가 | ❌ | 최대 3장 (가로 스크롤) |
| "광고" 라벨 | ❌ | ✅ (상단 노출 시) |

### 지도 뷰 (MVP-B)

| 항목 | 내용 |
|------|------|
| 전환 | 리스트 ↔ 지도 뷰 버튼 |
| 마커 | 지도 위 병원 위치 표시 |
| 마커 탭 | 병원 카드 미리보기 (하단 슬라이드업) |
| 지도 API | 카카오맵 / 네이버맵 **미정** |
| Flutter 패키지 | 카카오: `kakao_map_plugin` / 네이버: `flutter_naver_map` |

### 병원 상세 페이지

> 프리미엄 프로필 확장 상세는 → [12_revenue.md](./12_revenue.md) 참조

**무료 병원 프로필**

| 섹션 | 내용 |
|------|------|
| 기본 정보 | 병원명, 주소, 전화번호, 진료시간 |
| 전문 시술 분야 | 칩 최대 5개 |
| 병원 소개 | 최대 300자 |
| 병원 사진 | 최대 3장 |
| 의료진 | 대표 의료진 1명 |
| 위치 | 지도 미니뷰 + 주소 |
| 커뮤니티 태깅 후기 | 해당 병원이 태깅된 비포&애프터 피드 → [03_community.md](./03_community.md) |
| 출연자 후기 *(v2.0)* | 이 병원에서 시술받은 출연자의 스토리 → [12_cast_member.md](./12_cast_member.md) |
| 상담 요청하기 버튼 | 탭 시 → [02_consultation.md](./02_consultation.md) 상담 요청 플로우로 이동 |

**프리미엄 병원 프로필 (위 항목 + 확장) (MVP-B)**

| 확장 섹션 | 내용 |
|----------|------|
| 전문 시술 분야 | 칩 최대 10개 |
| 병원 소개 | 최대 1,000자 |
| 병원 사진 | 최대 10장 |
| 소개 영상 | 1개 (유튜브 임베드) |
| 의료진 프로필 | 전체 의료진 (경력 포함) |
| 시술 상세 설명 | 시술별 설명 + 이미지 |

### 홈 "이런 병원 어때요?" 섹션 (MVP-B)

> 수익 모델 상세는 → [12_revenue.md](./12_revenue.md) 참조

| 항목 | 내용 |
|------|------|
| 노출 위치 | 홈 화면 중간 — 유튜브 히어로 + 내 상담 현황 아래 |
| 형태 | 가로 스크롤 병원 카드 (최대 5개) |
| 라벨 | **"광고"** 라벨 표시 |
| 대상 | 프리미엄 병원 중 유저 관심 시술 분야 매칭 + 랜덤 로테이션 |
| 카드 탭 시 | 병원 상세 페이지로 이동 |
