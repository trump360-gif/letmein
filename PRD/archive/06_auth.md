## 일반 유저 가입 플로우

```
앱 실행 → 온보딩 (3장 스와이프) → 카카오 로그인
→ 이용약관 동의 (필수 3건 + 선택 2건)
→ 닉네임 설정 → 관심 시술 선택 (선택) → 메인화면
```

| 단계 | 상세 |
|------|------|
| 온보딩 | 3장 스와이프: ① 블랙라벨 소개 ② 코디네이터 매칭 설명 ③ 출연자 스토리 + 커뮤니티 소개 |
| 소셜 로그인 | MVP: 카카오 로그인 단독 / MVP-B: Apple 로그인 추가 (아래 상세) |
| 이용약관 동의 | 필수: 서비스 이용약관, 개인정보 수집·이용, 만 18세 이상 확인 |
| 선택 동의 | 마케팅 수신 동의, 민감정보(신체 사진) 수집 동의 |
| 닉네임 | 2~10자, 특수문자 제외, 중복 검사 |
| 관심 시술 | 칩 멀티셀렉 (스킵 가능), 추후 홈 피드 개인화에 활용 |

## Apple 로그인 *(v2.2 추가, MVP-B)*

> iOS 앱스토어 심사 필수 요건. 소셜 로그인을 제공하는 앱은 Apple Sign-in을 반드시 포함해야 함.

| 항목 | 내용 |
|------|------|
| 노출 조건 | iOS에서만 표시 (Android에서 미노출) |
| 위치 | 카카오 로그인 버튼 아래 |
| 인증 흐름 | Apple ID → 서버 identity token 검증 → JWT 발급 (카카오와 동일 흐름) |
| 기존 계정 연결 | 동일 이메일의 카카오 계정이 있으면 연동 안내 (선택) |
| 이름/이메일 | Apple은 최초 1회만 제공 — 서버에서 영구 저장 필수 |

**API**
```json
// POST /api/v1/auth/apple
// Req:  { "identityToken": "apple_jwt_token", "authorizationCode": "code", "fullName": "홍길동" }
// Res:  { "accessToken": "jwt", "refreshToken": "rt", "isNewUser": true, "user": { ... } }
```

**AC**
- [ ] iOS에서 Apple 로그인 버튼 노출
- [ ] Android에서 Apple 버튼 미노출
- [ ] Apple identity token 서버 검증 정상 동작
- [ ] 신규 가입 시 약관 동의 플로우 동일 적용
- [ ] 기존 카카오 계정과 동일 이메일 시 연동 안내

## JWT 인증 구조

> 기술 상세는 → [09_tech.md](./09_tech.md) 참조

```
카카오 로그인 → Go 서버에서 카카오 토큰 검증
→ 자체 JWT 발급 (access + refresh)
→ Flutter 앱 flutter_secure_storage에 저장
```

| 토큰 | 만료 | 용도 |
|------|------|------|
| Access Token | 15~30분 | API 요청 인증 |
| Refresh Token | 14일 | Access Token 재발급, Redis에 저장 |

- Refresh Token은 **Redis에 저장**하여 강제 로그아웃/탈퇴 시 즉시 무효화
- Centrifugo 접속 시에도 동일한 JWT로 인증 → [05_chat.md](./05_chat.md)

## 계정 탈퇴 프로세스

> ⚠️ 신체 사진 등 민감정보 포함. 탈퇴 시 완전 삭제 처리 필수. → [10_legal.md](./10_legal.md)

```
마이페이지 → 설정 → 계정 탈퇴 → 탈퇴 사유 선택 (선택)
→ 안내 문구 확인 → "탈퇴합니다" 텍스트 직접 입력 → 최종 확인 → 탈퇴 완료
```

| 항목 | 내용 |
|------|------|
| 탈퇴 유예 기간 | 7일 (유예 기간 내 로그인 시 탈퇴 철회) |
| 즉시 처리 | 로그인 차단, 프로필 비공개 전환 |
| 7일 후 처리 | 개인정보 완전 삭제 (닉네임, 사진, 상담 기록, 채팅 기록) → [08_media.md](./08_media.md) 사진 삭제 |
| 커뮤니티 게시물 | 작성자명 "탈퇴한 사용자"로 변환, 게시물 본문은 유지 (유저 선택에 따라 삭제 가능) → [03_community.md](./03_community.md) |
| 진행 중 상담 | 탈퇴 신청 시 진행 중 상담은 즉시 종료, 병원에 "유저가 탈퇴했습니다" 알림 → [05_chat.md](./05_chat.md) |
| 재가입 | 탈퇴 완료 후 동일 카카오 계정으로 신규 가입 가능 (이전 데이터 복구 불가) |
