> 상담 요청 사진, 비포&애프터 사진, 병원 사진 등 모든 이미지에 적용.

## 업로드 처리

```
클라이언트 촬영/선택 → EXIF 메타데이터 제거 (위치정보 등)
→ 클라이언트 사전 압축 (최대 2048px 리사이징)
→ 서버 업로드 (presigned URL 방식)
→ 서버 후처리 (비동기)
```

---

## 서버 후처리

| 처리 | 내용 |
|------|------|
| 리사이징 | 원본 보관 + 썸네일(300px) + 중간(800px) + 풀(2048px) 3종 생성 |
| EXIF 완전 제거 | GPS 좌표, 촬영 기기 정보 등 모든 메타데이터 삭제 |
| 포맷 변환 | WebP 변환 (CDN 전송 최적화) |
| 유해 콘텐츠 감지 | MVP: 미적용 / MVP-B: 자동 감지 도입 검토 |
