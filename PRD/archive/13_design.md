## UX 원칙

### 1. 최소 입력 (탭 선택 중심)

- 상담 접수: 텍스트 직접 입력 최소화, **탭/칩 선택** 중심
- 구식 신체 아이콘 배제
- 절차 최대한 간소화 (불필요한 Step 제거)

### 2. 프리미엄 큐레이션 대기 UX

상담 접수 후 대기 구간은 **배민 스타일 프로세스 UI**로 프리미엄 큐레이션 느낌 연출:

```
① 상담 접수 완료 ✅
② 코디네이터 검토 중 🔄
③ 추천 병원 선정 중 ⏳
④ 병원 매칭 완료 → 채팅 가능 ⏳
```

- 수평 프로그레스 바 + 각 단계 아이콘 애니메이션
- 대기 메시지: "블랙라벨의 전문 코디네이터가 고객님의 상담 내용을 꼼꼼히 검토하여, 가장 적합한 프리미엄 추천 병원을 엄선하고 있습니다"
- 응답 기준: 영업시간 내 2시간 이내. 야간/주말은 다음 영업일 오전 중 안내

### 3. 콘텐츠 이미지 가공

- 유튜브 썸네일을 **그대로 사용하지 않음**
- 플랫폼 톤앤매너에 맞게 **가공하여 노출** (다크 오버레이 + 타이틀 재배치)
- 이미지 위 텍스트는 항상 그림자/그래디언트 처리로 가독성 확보

### 4. 부드러운 트랜지션

- 화면 전환: fade + slide 복합 트랜지션
- 카드 탭: 미세한 스케일 애니메이션 (0.98 → 1.0)
- 로딩: 스켈레톤 UI (shimmer 효과, 다크 테마에 맞게)
- 스크롤: 히어로 영역 패럴랙스 효과

## 주요 컴포넌트 스타일

### 카드

```
배경: Surface 컬러 (#1E1E2E)
모서리: 16px radius
그림자: 없음 (다크 모드에서는 보더로 구분)
보더: 1px solid #2A2A3A
내부 패딩: 16px
```

### CTA 버튼

```
배경: Accent (#C0392B)
텍스트: White
모서리: 12px radius
높이: 52px
눌림 효과: 밝기 -10%
비활성: opacity 0.4
```

### 칩 (선택 태그)

```
기본: 보더만 (#2A2A3A), 텍스트 Secondary
선택됨: 배경 Accent, 텍스트 White
모서리: 20px radius (pill 형태)
```

### 바텀 네비게이션

```
배경: #0D0D0D
활성 아이콘: Accent (#C0392B)
비활성 아이콘: #666666
구분선: 상단 1px #2A2A3A
```
