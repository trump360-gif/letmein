## 서비스 비전

유튜브 **'렛미인'** 포맷의 변신 스토리 콘텐츠를 기반으로 유입된 팬덤을 자체 앱/웹 플랫폼으로 **락인(Lock-in)** 시키는 프리미엄 뷰티 콘텐츠·상담 플랫폼.

### 핵심 가치

- **탈(脫) 알고리즘 종속** — 유튜브 알고리즘 의존도에서 벗어나 자체 플랫폼 생태계 구축
- **투명한 정보 제공** — 영상으로 검증된 변신 콘텐츠 + 실제 경험 기반의 신뢰 있는 정보
- **코디네이터 매칭** — 전문 코디네이터가 검토 후 맞춤 프리미엄 병원을 연결

## 핵심 문제 정의

- 성형 시장의 정보 불균형 — 실제 후기보다 병원 마케팅 콘텐츠에 과다 노출
- 커뮤니티 파편화 — 카페·카톡·인스타 등 정보 채널 분산
- 비포&애프터 정보 부족 — 병원 직접 게시 불가(의료광고 규제)로 인한 정보 공백
- 유튜브 → 실행 전환 경로 부재 — 성형 콘텐츠 시청 후 병원 연결까지의 경로가 없음
- 병원 연결 과정의 불투명성 — 가격·시술 정보 비교 어려움

## 목표 사용자

| 구분 | 특성 | 핵심 니즈 |
|------|------|----------|
| 일반 유저 (소비자) | 성형 고민 중인 20–40대 여성 중심 | 신뢰할 수 있는 실제 후기, 맞춤 병원 추천, 커뮤니티 |
| 출연자 (Cast Member) | 렛미인 유튜브 출연 경험자 | 팬 커뮤니티 운영, 스토리 공유, 인플루언서 활동 |
| 병원 (B2B) | 강남권 중심 프리미엄 성형외과, 피부과 | 타겟 고객 확보, 프리미엄 채널 노출 |
| 코디네이터 | 상담 요청 검토 및 병원 매칭 전문 인력 | 효율적 매칭, 매칭 품질 관리 |
| CMS 관리자 | 플랫폼 운영팀 | 콘텐츠 검수, 유저/병원 관리, 출연자 관리 |

## 플랫폼 포지셔닝

**블랙라벨 — 유튜브 렛미인과 연결된 프리미엄 성형 상담 플랫폼**

> 한 줄 정의: "단순 병원 광고가 아닌 '영상으로 검증된 변신 콘텐츠'로 유저의 신뢰를 확보하고, 코디네이터가 엄선한 프리미엄 병원을 매칭하는 콘텐츠-커뮤니티 버티컬 플랫폼"

커뮤니티 + 출연자 스토리가 서비스의 핵심 가치이며, 코디네이터 매칭이 수익화 연결 고리.

## 경쟁 환경 및 차별화

| 기존 플랫폼 | 특징 | 블랙라벨의 차별점 |
|------------|------|-----------------|
| 강남언니 | 병원 리스팅 + 리뷰 + 광고 중심 | 코디네이터 큐레이션 + 유튜브 출연자 검증 콘텐츠 |
| 바비톡 | 커뮤니티 + 병원 정보 | 출연자 팬덤 락인 + 프리미엄 포지셔닝 |
| 여신티켓 | 할인·쿠폰 중심 | VIP 매칭 구조 (소수 프리미엄 병원 집중) |

### 차별화 근거

1. **신뢰 생성의 주체 변화** — 병원 광고 → 유튜브 영상 기반 검증된 출연자 스토리
2. **프리미엄 큐레이션** — 초기 강남권 프리미엄 병원 약 10여 곳만 세팅, 코디네이터가 맞춤 매칭
3. **지속적 커뮤니티 락인** — 유튜브 팬덤이 출연자 스토리 피드와 상호작용하며 체류
4. **중장기 생태계** — WorkB EMR/CRM 연동, 일본 인플루언서 인프라를 통한 글로벌 확장

## 수익 모델 (확정)

> 상세 명세는 → [12_revenue.md](./12_revenue.md) 참조

### 핵심 원칙: "매칭은 무료, 수익은 광고에서만"

의료법 제27조(환자 유인·알선 금지) 분석 결과, **매칭 기능과 과금을 완전 분리**하는 구조를 채택.

- **모든 승인된 병원**은 역경매 열람·상담 발송·채팅에 **무료 참여**
- 과금은 **프로필 정보량 확장**과 **광고 노출 위치**에만 연결
- 유료 노출에는 반드시 **"광고" 라벨** 표시

### 수익원

| 상품 | 과금 | 시점 |
|------|------|------|
| 프리미엄 입점 | 월정액 — 상담 카드 정보 확장, 검색 상단 노출, 프로필 확장 | MVP-B |
| 네이티브 광고 | 충전형 CPM — 커뮤니티 피드 콘텐츠형 광고 | 3차 |

### 영구 제외 항목

- 매칭·상담 건당 수수료 (CPA) → **영구 제외** (의료법 27조 위반 확실)
- 과금 병원만 역경매 참여 가능한 구조 → **영구 제외** (알선 대가 해석)
- 유저 과금 → 현재 미검토

### 중장기 수익 모델

| 분야 | 내용 |
|------|------|
| B2B 데이터 리포트 | 비식별 통계 기반 시술 수요 트렌드 |
| 유료 멤버십 | 프리미엄 구독 모델 |
| 커머스 | 뷰티/의료 관련 상품 판매 |
| 뷰티 의료 관광 | 일본 등 해외 시장 확장 |
| WorkB 연동 | EMR/CRM 병원 운영 솔루션 연동 |
