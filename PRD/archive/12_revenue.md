## 확정 배경

의료법 제27조(환자 유인·알선 금지) 분석 결과, 플랫폼이 병원에서 대가를 받으면서 환자를 연결하는 구조는 **"영리 목적의 알선"**으로 해석될 수 있다.

이를 회피하기 위해 **"매칭 기능과 과금을 완전 분리"**하는 구조를 채택한다.

| 검토한 모델 | 판단 | 사유 |
|------------|------|------|
| 매칭 건당 수수료 (CPA) | **영구 제외** | 의료법 27조 알선 대가 — 위반 확실 |
| 정보열람 과금 (열람료) | **제외** | 열람 = 매칭 전 단계 → 실질적 알선 대가 해석 가능 |
| 입점비에 매칭 접근권 포함 | **제외** | 과금 병원만 상담 가능 → 입점비 = 알선 대가 |
| **프리미엄 프로필 + 광고 노출** | **채택** | 매칭과 무관한 순수 광고 상품 — 강남언니 등 검증된 구조 |

---

## 핵심 원칙

> **"매칭은 무료, 수익은 광고에서만"**

1. **모든 승인된 병원**은 역경매 열람·상담 발송·1:1 채팅에 **무료 참여** 가능
2. 과금은 오직 **프로필 정보량 확장**과 **광고 노출 위치**에만 연결
3. 과금 여부와 관계없이 역경매 선착순 5개 슬롯은 **동일 조건** (도착 시간순)
4. 무료 병원도 **충분히 경쟁 가능한 수준**의 기본 정보 제공
5. 유료 노출에는 반드시 **"광고" 라벨** 표시 (의료법 56조 + 표시광고법)
6. 계약서 명목은 **"광고 게재 서비스"** — "소개/알선/매칭" 용어 절대 금지

## 상품 1: 프리미엄 입점 (월정액)

> 계약 명목: **"플랫폼 프로필 광고 게재 서비스 이용 계약"**

### 역경매 상담 카드 — 무료 vs 프리미엄

유저가 받은 상담 카드에서 프리미엄 병원은 더 풍부한 정보를 표시한다.
카드 정렬은 **도착 시간순으로 동일** — 프리미엄이라고 상단 배치되지 않음.

| 항목 | 무료 병원 | 프리미엄 병원 |
|------|----------|-------------|
| 병원명 + 대표 이미지 | 1장 | 1장 |
| 전문 분야 칩 | 최대 3개 | 최대 3개 |
| 시술 경력 | 한 줄 | 한 줄 |
| 상담 메시지 | 전문 | 전문 |
| 상담 가능 방식 | 아이콘 | 아이콘 |
| 병원 사진 추가 | ❌ | ✅ 최대 5장 |
| 의료진 프로필 | ❌ | ✅ 의료진명 + 경력 |
| 소개 영상 썸네일 | ❌ | ✅ |
| 시술 사례 건수 | ❌ | ✅ (예: "코성형 1,200건+") |

> 상세 UI 명세는 → [02_auction.md](./02_auction.md) 참조

### 병원 탐색 상단 노출

프리미엄 병원은 병원 탐색 검색 결과에서 **상단에 노출**된다.

| 항목 | 내용 |
|------|------|
| 노출 위치 | 검색 결과 최상단 (일반 결과 위) |
| 라벨 | **"광고"** 라벨 필수 표시 |
| 최대 노출 수 | 검색당 프리미엄 병원 최대 3개 |
| 노출 기준 | 프리미엄 병원 간 — 시술 분야 일치도 + 랜덤 로테이션 |
| 일반 병원 | 프리미엄 영역 아래에 기존 정렬 기준으로 노출 |

> 상세 UI 명세는 → [04_hospital.md](./04_hospital.md) 참조

### 프로필 페이지 확장

| 항목 | 무료 병원 | 프리미엄 병원 |
|------|----------|-------------|
| 기본 정보 (병원명, 주소, 전화, 진료시간) | ✅ | ✅ |
| 전문 시술 분야 | 칩 최대 5개 | 칩 최대 10개 |
| 병원 소개 텍스트 | 최대 300자 | 최대 1,000자 |
| 병원 사진 | 최대 3장 | 최대 10장 |
| 소개 영상 | ❌ | ✅ 1개 (유튜브 임베드) |
| 의료진 프로필 | 대표 의료진 1명 | 전체 의료진 |
| 시술 상세 설명 | ❌ | ✅ 시술별 설명 + 이미지 |
| 커뮤니티 태깅 후기 | ✅ | ✅ |

> 상세 UI 명세는 → [04_hospital.md](./04_hospital.md) 참조

### 홈 "이런 병원 어때요?" 섹션

| 항목 | 내용 |
|------|------|
| 노출 위치 | 홈 화면 중간 — 유튜브 히어로 + 내 상담 현황 아래 |
| 형태 | 가로 스크롤 병원 카드 (최대 5개) |

## 상품 2: 네이티브 광고 (충전형 CPM)

> 계약 명목: **"플랫폼 디스플레이 광고 게재 서비스 이용 계약"**

### 커뮤니티 피드 네이티브 광고

병원이 커뮤니티 피드에 콘텐츠형 광고 카드를 게시한다.
일반 게시물과 **동일한 카드 포맷**이되, "광고" 라벨로 구분.

| 항목 | 규칙 |
|------|------|
| 삽입 비율 | 일반 게시물 **5개당 광고 1개** |
| 연속 광고 | **금지** — 광고끼리 연달아 노출 안 됨 |
| 라벨 | **"광고"** 라벨 필수 |
| 카드 포맷 | 병원명 + "광고" 라벨 + 이미지 + 텍스트 (60자 이내) + "병원 프로필 보기" 버튼 |
| 좋아요/댓글 | **없음** — 깔끔한 브랜딩 유지, 악성 댓글 방지 |
| 소재 심사 | CMS에서 운영팀 사전 승인 후 게재 → [07_cms.md](./07_cms.md) |
| 의료광고 규정 | 의료법 56조 의료광고 규정 준수 필수 → [10_legal.md](./10_legal.md) |

> 상세 UI 명세는 → [03_community.md](./03_community.md) 참조

### 충전형 과금 구조

| 항목 | 내용 |
|------|------|
| 크레딧 충전 | 최소 5만원 단위 |
| 과금 방식 | **CPM** (1,000회 노출당 과금) |
| 캠페인 설정 | 지면 선택, 기간, 일 예산 한도, 소재 등록 |
| 소진 시 | 자동 중단 + 알림 |
| 잔액 환불 | 미소진 크레딧 환불 가능 (수수료 없음) |
| 성과 리포트 | 노출수, 프로필 보기 클릭수, CTR |

### 캠페인 관리 (병원 앱 내)

```
광고 관리 화면:

┌──────────────────────────────────┐
│  광고 크레딧: 150,000원           │
│  [충전하기]                      │
│                                  │
│  [캠페인 만들기]                  │
│  ├── 지면: 커뮤니티 피드          │
│  ├── 기간: 3/25 ~ 3/31          │
│  ├── 일 예산: 30,000원           │
│  └── 소재: 이미지 + 문구         │
│                                  │
│  진행 중 캠페인                   │
│  └── 피드광고 | 노출 12,340      │
│       클릭 234 | 소진 45,000원    │
└──────────────────────────────────┘
```

## 절대 금지 과금 구조

| 금지 항목 | 사유 |
|-----------|------|
| 매칭·상담 건당 수수료 (CPA/CPS) | 의료법 27조 알선 대가 — 위반 확실 |
| 과금 병원만 역경매 열람/발송 가능 | 입점비 = 매칭 접근권 → 알선 대가 해석 |
| 과금 병원 상담 카드 우선 정렬 | 매칭 기능에 과금 차등 → 알선 해석 가능 |
| 유저 개인정보 병원 직접 전달 | 알선 행위의 핵심 요소 |
| 채팅 기능 과금 차등 | 매칭 기능에 해당 |
| "소개비", "알선비", "매칭비" 명목 과금 | 계약서에 해당 용어 사용 금지 |

## 수익화 로드맵

| 단계 | MAU 구간 | 수익원 | 예상 |
|------|---------|--------|------|
| MVP-A | 0~2,000 | 없음 (무과금) | 유저·병원 확보에 집중 |
