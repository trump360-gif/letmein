## 비포&애프터 게시판 (MVP-A)

> ⚠️ 유저 자율 커뮤니티로 운영. 플랫폼은 인센티브를 제공하지 않으며, 자연 발생 콘텐츠 구조로 설계. (의료광고 간접 유도 리스크 해소)
> 법적 근거: [10_legal.md](./10_legal.md) — 비포&애프터 간접 광고 리스크

**운영 원칙**
- 유저가 자발적으로 시술 전후 사진 + 후기 게시
- **플랫폼 인센티브 없음** — 포인트·리워드 미제공
- 병원 태깅 허용 (유저 자발적 선택, 강제 아님) → 탭 시 [04_hospital.md](./04_hospital.md) 병원 상세로 이동
- 태깅된 병원은 해당 게시물 승인·거부 불가 (유저 주도권 유지)
- 신고 기능 → [07_cms.md](./07_cms.md) 검수 후 블라인드

**게시물 구성**
- 시술 카테고리 태그 (필수)
- 시술 시기 입력 (선택)
- 전후 사진 (선택, 최대 10장) → [08_media.md](./08_media.md) 공개 버킷 저장
- 후기 텍스트 (최소 20자)
- 병원 태깅 (선택)

**초기 콘텐츠 확보 전략**

> 자율 커뮤니티 모델이므로 인센티브 없이 초기 콘텐츠를 확보하는 전략이 필요.

| 전략 | 내용 |
|------|------|
| 시드 콘텐츠 | 운영팀이 실제 시술 경험자 10~20명 섭외하여 초기 후기 게시 요청 (무보상, 순수 자발) |
| 커뮤니티 이전 유도 | 기존 성형 카페·커뮤니티 활동 유저 대상 앱 안내 |
| 유튜브 채널 연동 | 운영사 유튜브 시청자를 앱 커뮤니티로 유도 |
| Q&A 시드 | 운영팀이 자주 묻는 질문 20~30개 사전 등록 |

## Before/After 슬라이더 비교 뷰 *(v2.2 추가, MVP-B)*

> 비포&애프터 게시글에서 전후 사진을 인터랙티브하게 비교하는 킬러 UX.

| 항목 | 내용 |
|------|------|
| 적용 대상 | board_type = `before_after` 게시글 중 사진 2장 이상 |
| UI | 좌우 드래그 슬라이더 — 왼쪽 Before / 오른쪽 After |
| 사진 지정 | 게시글 작성 시 사진별 "Before" / "After" 태그 선택 |
| 상세 화면 | 슬라이더 비교 뷰 (기본) + 일반 갤러리 뷰 토글 |
| 목록 화면 | 썸네일에 Before/After 분할 미리보기 |
| 미로그인/설정 | 민감정보 동의 미완료 시 사진 블러 처리 |

**AC**
- [ ] 게시글 작성 시 사진에 Before/After 태그 지정 가능
- [ ] 상세 화면에서 슬라이더 드래그로 전후 비교
- [ ] 슬라이더 ↔ 일반 갤러리 뷰 토글
- [ ] 민감정보 미동의 시 블러 처리
- [ ] Before/After 태그 없는 사진은 일반 갤러리로 표시

## 출연자 스토리 Q&A 피드 *(v2.0 신규)*

> 상세 시스템 명세: → [12_cast_member.md](./12_cast_member.md)

렛미인 유튜브 출연자가 자신의 변신 스토리, 회복 과정, Q&A를 공유하는 전용 피드.

### 피드 구성

- 출연자만 스토리 업로드 가능 (인증 배지 보유자)
- 사진/영상 링크 + 텍스트 + 유튜브 회차 링크 자동 고정
- 유저가 댓글로 질문 → 출연자가 "출연자 답변" 배지와 함께 답변
- 출연자 답변은 댓글 상단 고정 옵션

### 홈 연동

- 홈 상단에 **인스타그램 스타일 스토리 바** (원형 프로필) 표시
- 새 스토리가 있는 출연자: 레드/버건디 테두리 → [13_design.md](./13_design.md)
- 팔로우 중인 출연자 우선 정렬
- 팔로우 시 새 스토리 업로드 푸시 알림

### Phase

- MVP-A: 출연자 인증/배지만
- MVP-B: 스토리 피드 전체 기능 (팔로우, Q&A, 홈 스토리 바)

## 콘텐츠 운영 정책 *(v2.0 갱신)*

### 자동 차단 (게시 불가)

게시글 작성 시 아래 항목 포함 시 게시 차단 + 안내 메시지 표시:

- 병원명 / 의원명
- 특정 의사 이름
- 타 플랫폼 링크 (강남언니, 바비톡 등)

### 자동 블라인드 `***` 처리

게시글은 올라가되 해당 단어만 블라인드 처리:

- 욕설 / 비방
- 가격 정보 직접 언급 ("150만원에 했어요" 등)

### 관리자 승인 후 게시

- 수술 전후 극단적 신체 노출 사진

> 금지어 관리: CMS 금지어 관리 → [07_cms.md](./07_cms.md)

## 네이티브 광고 (3차)

> 수익 모델 상세는 → [12_revenue.md](./12_revenue.md) 참조
> 광고 소재 심사는 → [07_cms.md](./07_cms.md) 참조
> 의료광고 규정은 → [10_legal.md](./10_legal.md) 참조

병원이 커뮤니티 피드에 콘텐츠형 광고 카드를 게시한다. 일반 게시물과 동일한 카드 포맷이되, "광고" 라벨로 구분.

### 삽입 규칙

| 항목 | 규칙 |
|------|------|
| 삽입 비율 | 일반 게시물 **5개당 광고 1개** |
| 연속 광고 | **금지** — 광고끼리 연달아 노출 안 됨 |
| 라벨 | **"광고"** 라벨 필수 표시 |
| 좋아요/댓글 | **없음** — 브랜딩 유지 + 악성 댓글 방지 |
| 소재 심사 | CMS에서 운영팀 사전 승인 후 게재 |
| 과금 | 충전형 CPM (1,000회 노출당 과금) |

### 광고 카드 포맷

```
┌──────────────────────────────────┐
│  🏥 병원명 · 광고                 │
│  "#시술분야 광고 문구 (60자 이내)" │
│  [이미지]                        │
│  병원 프로필 보기 →               │
└──────────────────────────────────┘
```

| 항목 | 규격 |
|------|------|
| 병원명 | 자동 표시 |
| "광고" 라벨 | 필수, 병원명 우측 |
| 광고 문구 | 최대 60자 |
| 이미지 | 1장, 일반 게시물과 동일 비율 |
| CTA 버튼 | "병원 프로필 보기" → [04_hospital.md](./04_hospital.md) 병원 상세 |
| 의료광고 규정 | 의료법 56조 준수 — 과대 광고, 시술 전후 사진 금지 |
