## 4. 콘텐츠 관리

### 4-1. 게시글 관리

- **경로**: `/contents/posts`

#### 글 목록

- 상태별 필터: 임시저장, 발행, 숨김, 삭제, 블라인드, 예약
- 게시판별, 글 유형별(TREND/EVERGREEN), 언어별(KO/JA), AI생성 여부 필터
- 검색, 페이지네이션

#### 일괄 작업

- 선택한 글 일괄 발행 / 블라인드 / 삭제 / 게시판 이동

#### 글 작성/수정

1. "글 작성" 버튼 클릭
2. 제목, 본문, 게시판 선택
3. 상태 설정 (임시저장 / 발행 / 예약발행)
4. SEO 메타 정보 입력 (메타 타이틀, 디스크립션, OG 이미지)
5. 저장

#### SEO 점수

- SEO / AEO / GEO 점수가 자동 산출됩니다.

### 4-2. 게시판 관리

- **경로**: `/contents/boards`
- 게시판 생성/수정/삭제
- 계층 구조 (상위/하위 게시판)
- 게시판 그룹핑
- 스킨 설정: 리스트뷰, 카드뷰, 앨범뷰
- 접근 권한 설정 (읽기/쓰기/댓글/업로드 등급)
- 익명/비밀글 허용, 좋아요/싫어요/댓글/답글 기능 ON/OFF
- 신고 임계값 및 자동 블라인드 설정
- 순서 변경 (드래그 앤 드롭)

### 4-3. 댓글 관리

- **경로**: `/members/comments`
- 전체 댓글 조회
- 상태별 필터, 검색
- 블라인드 / 삭제 처리

## 5. 자동 글쓰기 (Auto-Posting)

- **경로**: `/auto-posting`

### 설정 생성 (5단계 마법사)

| 단계 | 내용 |
|------|------|
| 1단계 | **콘텐츠 소스 선택** — sungyesa, girlschannel, cosme 중 선택 |
| 2단계 | **페르소나 선택** — BLOG 또는 COMMUNITY 타입, 콘텐츠 작성자 지정 |
| 3단계 | **글 유형 & 언어** — TREND/EVERGREEN, KO/JA |
| 4단계 | **스케줄 설정** — 설정명, 대상 게시판, 일일 생성 수, 생성 시간, 발행 시간대, 일일 발행 한도, 이미지 생성/자동번역/자동발행 토글 |
| 5단계 | **SEO 임계값** — 최소 SEO/AEO/GEO 점수 설정 |

### 설정 관리

- 설정 목록 조회
- 활성/비활성 토글
- 점수 임계값 수정
- 콘텐츠 작성자 지정
- 삭제

### 배치 실행

- "실행" 버튼으로 수동 배치 실행
- 실행 중인 배치 중지
- 콘텐츠 소스 동기화 (외부 소스에서 최신 콘텐츠 가져오기)

### 실행 로그

- 배치 실행 이력 조회
- 상태: SUCCESS / FAILED / SKIPPED
- SEO/AEO/GEO 점수 확인
- 로그에서 직접 발행 가능
- 실행 중 실시간 폴링

## 6. 페르소나 관리

- **경로**: `/personas`
- AI 글쓰기 페르소나 생성/수정/삭제
- 타입: BLOG / COMMUNITY
- 설정 항목: 이름, 타입, 설명, 글쓰기 스타일 가이드, 아바타 URL, 활성 상태
- 타입별 필터, 페이지네이션 (20개씩)

## 7. 회원 관리

- **경로**: `/members`

### 회원 목록

- 이메일/닉네임 검색
- 등급별, 상태별, 가입일 범위 필터
- 정렬: 가입일, 최근 로그인, 포인트, 등급

### 회원 상세

- 기본 정보 (이메일, 닉네임, 전화번호, 소셜 로그인 정보)
- 등급 / 포인트 / 상태
- 가입일 / 최근 로그인

### 회원 관리 기능

| 기능 | 설명 |
|------|------|
| 포인트 관리 | 포인트 지급 / 차감 |
| 등급 변경 | 등급 업그레이드 / 다운그레이드 |
| 정지 | 회원 정지 처리 |
| 정지 해제 | 정지 상태 해제 |
| 강제 로그아웃 | 즉시 세션 만료 |

### 회원 상태

- 일반(active), 휴면(dormant), 정지(suspended), 삭제(deleted)

## 8. 운영 관리

### 8-1. 메뉴 관리

- **경로**: `/operations/menus`
- GNB, 푸터 등 네비게이션 메뉴 관리
- 트리 구조 (상위/하위 메뉴)
- 링크 타입 (내부/외부), 아이콘, 뱃지 설정
- 등급 제한 설정
- 드래그 앤 드롭 순서 변경

### 8-2. 배너 관리

- **경로**: `/operations/banners`
- 프로모션 배너 생성/수정/삭제
- 디바이스별 이미지 (PC/모바일/태블릿)
- 위치, 링크 URL, UTM 캠페인 설정
- 타겟 설정 (전체, 신규 회원, 등급별)
- 노출 기간 예약
- 닫기 옵션 (오늘 하루, 1일, 1주, 영구)
- A/B 테스트 (그룹, 비율)
- 노출/클릭 통계 조회

### 8-3. 팝업 관리

- **경로**: `/operations/popups`
- 이미지/HTML 팝업 생성/수정/삭제
- 노출 범위 (전체, 특정 게시판)
- 크기, 위치, 애니메이션 설정
- 타겟 및 등급 제한
- A/B 테스트, 최대 노출 횟수
- 노출 기간 예약

### 8-4. 금칙어 관리

- **경로**: `/operations/banned-words`
- 금칙어 등록/수정/삭제
- 패턴 타입: 일치, 정규식
- 게시판별 적용
- 매칭 시 처리: 별표 치환, 차단, 경고
- 테스트 기능

### 8-5. 신고 관리

- **경로**: `/operations/reports`
- 사용자 신고 목록 조회 (글/댓글/회원)
- 상태별 필터: 대기, 승인, 거부, 완료
- 신고 처리: 승인(제재 적용) / 거부

### 8-6. 제재 관리

- **경로**: `/operations/sanctions`
- 제재 이력 조회
- 제재 유형: 경고, 제한, 정지, 차단
- 활성/해제 상태 필터
- 제재 해제 기능

## 9. 분석 / 설정

### 9-1. 분석 (Analytics)

- **경로**: `/config`
- 기간 선택: 7일 / 30일 / 90일 / 직접 입력

| 탭 | 내용 |
|----|------|
| 회원 통계 | 신규 가입, 활성 사용자, 성장 추이 |
| 게시글 통계 | 신규 글, AI 생성 vs 수동, 게시판별 분포, 평균 SEO 점수 |
| 댓글 통계 | 신규 댓글, 활동 추이 |
| 배너 통계 | 노출수, 클릭률, 디바이스별 분석 |
| 신고 통계 | 신고 수, 유형별, 처리율 |
| 알림 통계 | 발송 수, 성공률, 채널별 |
| 퍼널 분석 | 전환 퍼널 시각화, 사용자 여정 |

- CSV/Excel 내보내기 지원

### 9-2. 시스템 관리

- **경로**: `/config/system`

| 탭 | 내용 |
|----|------|
| 활동 로그 | 관리자 액션 로그, 모듈/액션별 필터, 내보내기 |
| 로그인 이력 | 관리자 로그인 기록, IP/UA 추적, 새 IP 감지 |
| 역할 관리 | 관리자 역할 생성/수정, 모듈별 권한 매트릭스 (읽기/쓰기/삭제) |
| 관리자 목록 | 관리자 계정 조회, 역할 변경, IP 화이트리스트, 2FA 상태 |
| 시스템 로그 | 시스템 이벤트, 에러 추적 |

### 9-3. 사이트 설정

- **경로**: `/config/settings`

| 탭 | 내용 |
|----|------|
| 기본 정보 | 사이트명, 설명, 로고, 연락처, SNS |
| SEO 설정 | 기본 메타태그, OG 이미지, 구조화 데이터, SEO 스코어링 규칙 |
| 이용약관 | 약관 버전 관리, 필수/선택, 시행일 |
| 보안 | 비밀번호 정책, 2FA, IP 화이트리스트, 세션 타임아웃 |
| 환경 설정 | API 엔드포인트, 기능 플래그, 디버그 모드 |
| 다국어 | 번역 키 관리 (KO/JA/EN), 내보내기/가져오기 |

### 9-4. API 키 관리

- **경로**: `/config/api-settings`

| 탭 | 내용 |
|----|------|
| 소셜 로그인 | Google, Naver, Kakao, Facebook OAuth 설정 |
| 알림 API | 이메일(SMTP), SMS, 카카오톡, FCM 설정 |
| 결제/본인인증 | PG사, 본인인증 설정 |
| 기타 | 웹훅, 서드파티 API |

### 9-5. 알림 관리

- **경로**: `/config/notifications`

| 탭 | 내용 |
|----|------|
| 수동 발송 | 대상 선택, 채널 선택 (인앱/이메일/SMS/카카오), 예약 발송 |
| 이메일 템플릿 | 템플릿 조회/수정, 변수 치환, 테스트 발송 |
| 발송 대기열 | 대기 중 알림 조회, 우선순위, 재시도 |
| 발송 이력 | 발송 내역, 전달 상태, 필터 |
| 웹훅 | 웹훅 엔드포인트 관리, 이벤트 구독, 테스트 |
| 알림 통계 | 채널별 발송 수, 성공률, 사용자 참여 |

## 웹 사이트 (공개 페이지)

어드민에서 설정한 내용이 반영되는 공개 웹사이트입니다.

- **홈페이지**: 어드민에서 구성한 동적 섹션이 순서대로 렌더링
- **게시판 목록**: `/[slug]` — 게시판별 글 목록
- **글 상세**: `/[slug]/[postId]` — 글 본문 + 댓글
- 반응형 디자인 (모바일/태블릿/데스크톱)
- SEO 메타태그 자동 적용
